KEEPCUBE

SMART HOME SOLUTION FOR DIYers

What is it?

KEEPCUBE is an open-source smart home solution, which is easy to use and allows anyone to automate his home.

With fully 3D printable chassis and build schematics available online, every maker will be able to build his own CUBE.

Pure
wireless

KEEPCUBE is using wireless technology to communicate with smart accessories, that means there is no need to physicaly modify the structure of your flat.

Create and share
compatibility

Not only you can teach your CUBE to be compatible with new devices. But you can also share this compatibility with other KEEPCUBE users.

Design your own
smart accessories

You can easily connect your own devices to the CUBE by using our web user interface.

With ordinary
components

KEEPCUBE is built to be compatible with usual and available components, so every user can achieve more while paying less.

Powerful user
interface

Web user interface offers you many features. Including automatition, adding new device compatibility, monitoring your home and many others.
